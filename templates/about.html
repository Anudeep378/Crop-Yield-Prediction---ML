About the Project

What is
CropSense?

CropSense is an end-to-end machine learning web application that forecasts agricultural crop yield across Indian states — combining historical data, live climate feeds, and a trained Random Forest model into a single, accessible tool for farmers, agronomists, and researchers.

🌱 Motivation

India's agriculture sector feeds over a billion people, yet yield uncertainty remains one of the biggest risks for small-scale farmers. Unpredictable rainfall, shifting seasons, and choosing the wrong crop for a region can all result in devastating losses.

CropSense was built to close this gap — making data-driven yield prediction available through a simple web form, with no statistical expertise required. If the model detects a low-yield scenario, it immediately suggests better-suited crops based on historical regional performance.

🧠 The ML Model

The predictive core of CropSense is a Random Forest Regressor trained on the Indian Crop Production dataset. The model uses 200 estimators and was trained with an 80/20 train-test split.

📋 Model Input Features

The model accepts eight input features covering the crop, location, climate, and farm resources.

Feature Type Description
State Categorical Indian state where the crop is grown
Crop Categorical Crop variety (e.g., Rice, Wheat, Cotton)
Season Categorical Growing season (Kharif, Rabi, Whole Year…)
Crop Year Numeric Harvest year
Area Numeric Cultivated area in hectares
Annual Rainfall Auto-fetched Rainfall in mm — filled from live weather API
Fertilizer Numeric Fertilizer applied (kg)
Pesticide Numeric Pesticide applied (kg)

✨ Key Features

🌤️
Live Weather Auto-fill
When you select a state, CropSense fetches real-time weather for the state capital via Open-Meteo and pre-fills the rainfall field for you.
💡
Smart Crop Recommendations
If predicted yield falls below the 25th percentile, the app surfaces the top 5 historically high-yield crops for your region and season.
📊
Regional Data Intelligence
State-level average rainfall is pre-computed from the dataset and used as a fallback when the weather API is unavailable.
🔒
No API Key Needed
Open-Meteo is a completely free, open-source weather API with no registration or key required — ever.
